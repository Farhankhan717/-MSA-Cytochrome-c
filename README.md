# Task 2: Multiple Sequence Alignment of Cytochrome c

## Overview
Multiple Sequence Alignment (MSA) of Cytochrome c protein 
across 5 evolutionarily diverse species using Clustal Omega.

## Species Analyzed
| Species | Common Name | Accession |
|---------|-------------|-----------|
| Homo sapiens | Human | P99999 |
| Mus musculus | Mouse | P62897 |
| Gallus gallus | Chicken | P00007 |
| Danio rerio | Zebrafish | Q7T3A0 |
| Saccharomyces cerevisiae | Yeast | P00044 |

## Tools Used
- NCBI Protein Database — sequence retrieval
- Clustal Omega (EBI) — multiple sequence alignment
- Jalview — visualization

## Files
- cytochrome_sequences.fasta — input sequences
- cytochrome_aligned.aln — alignment output
- Task2_MSA_YourName.pdf — full report

## Key Finding
The CXXCH heme-binding motif is fully conserved 
across all 5 species confirming strong purifying 
selection on Cytochrome c throughout evolution.
